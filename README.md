# PERSONAL-PORTFOLIO
### 실행 방법
레포지토리 내 모든 파일을 받아 VSC에서 Live server 설치 후 **index.html** 실행

<br><br>

## 1. 프로젝트 개요
프론트엔드 공부를 위해 개발한 첫 프론트엔드 프로젝트입니다.

<br>

## 2. 프로젝트 설명
**포트폴리오** 목적의 원 페이지 웹 사이트입니다. 필수적인 내용들만 간추려 작성했습니다.

<br>

## 3. 사용 기술
- HTML
- CSS
- VanilaJS
