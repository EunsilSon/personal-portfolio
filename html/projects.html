One-Pass

2021.03 - 2021.09 (5人 팀 프로젝트)

졸업 작품으로 개발한 AI 기반의 문제 풀이를 제공하는 웹 사이트입니다. 문제은행식의 CCNA 문제가 출제되는 학교 시험을 대비하기 위한 서비스입니다. 서버 구축과 CI/CD 구현에 집중했습니다.
BE는 서비스들이 여러 개로 분리되어 Eureka 서버를 기준으로 연결되는 마이크로 서비스 아키텍처를 경험해보았습니다. 분산된 서비스들을 하나의 API로 상호작용하는 과정에서 라우팅과 로드밸런싱의 개념을 알게되었습니다.
빌드로 인한 서버 부하 방지를 위해 서버를 분리해 CI 서버와 CD 서버의 각 역할을 쉘 스크립트로 조작 및 원격으로 제어했습니다.
CI 서버: Jenkins를 GitHub Webhook과 연결해 자동 빌드 수행 후 성공 시 Docker image 생성 후 Docker Hub에 push 했습니다.

CD 서버: Docker Hub에 있는 image를 pull 받아 docker 컨테이너를 생성해 자동 배포를 수행했습니다.

원하는 개수만큼 문제 풀기, 문제의 핵심 키워드 보기, 포인트 기능

2020 이브와 ICT : 스마트 제어 수직 형 스마트 팜

2020.05 - 2020.11 (5人 팀 프로젝트)

2020 이브와 ICT 공모전에서 은상을 수상한 '수직 파이프 스마트팜' 입니다. 스마트팜은 사물인터넷(IoT)을 통해 작물 성장에 필요한 데이터를 수집하여 작물을 재배할 수 있는 농장입니다. PVC 파이프로 제작하고 각종 센서를 부착해 환경을 구성했습니다.
개발 실력보다 팀원들과의 의사소통의 중요성을 배움. 개개인이 여러 의견을 내어 원활한 소통을 통한 협동심이 큰 성과를 내기에 가장 좋은 거름이라는 것을 몸소 느낌
개발 과정들을 문서화하는 과정에서 유지 보수를 위해 객관적이고, 가독성 좋은 글을 써 개발 과정을 설명하는 것도 개발만큼 중요한 부분임을 깨달음
1, 2, 3차 개발보고서와 제작 설계서를 작성하여 통과했고, 최종 발표를 거쳐 최종적으로 은상이라는 성과를 이뤄냄
