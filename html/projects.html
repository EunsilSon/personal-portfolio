Linux 환경의 JSP 웹 사이트 구축

2020.12 (1人 개인 프로젝트)

‘Linux 환경에서 JDK+TOMCAT+MYSQL 을 연동한 사이트 구축’ 과제를 받아 수행했습니다. Ubuntu 운영체제의 터미널에서 작업을 진행했고 JDK, Tomcat8, MySQL을 설치 및 연동해 웹 서버를 구축했습니다.
샘플 페이지 코드를 실행시켜 JDBC 연동, 페이지 실행 및 DB 접근 테스트를 진행했습니다.
CMD 환경에 익숙해지고 리눅스의 폴더 구조를 몸소 느낄 수 있었던 과제입니다.
VMware에서 가상 OS를 설치해 진행했으며 '일단 부딪혀보자' 라는 생각에 여러 명령들을 입력해보고 무언가 꼬였다는 느낌이 들면 가상 OS를 재설치하여 처음부터 다시 작업해보면서 리눅스에 흥미를 갖게되었습니다.

2020 이브와 ICT : 스마트 제어 수직 형 스마트 팜

2020.05 - 2020.11 (5人 팀 프로젝트)

2020 이브와 ICT 공모전 은상을 수상한 '수직 파이프 스마트팜' ICT 프로젝트 입니다. ‘스마트 팜’이란 사물인터넷(IoT) 을 통해 데이터를 수집해 사용자들이 공간 및 날씨에 구애받지않고 취적의 생육 환경에서 작물을 쉽게 키울 수 있도록 하는 농장입니다. 해당 프로젝트의 스마트 팜은 철제 망에 PVC 파이프를 수직으로 각 파이프들을 호스로 연결시킨 후 각종 센서를 부착해 환경을 구성했습니다.
팀장을 맡아 전반적인 업무 총괄 및 보고서 작성을 맡았습니다. 이 프로젝트를 진행하며 팀원들과의 의사소통과 협동심의 중요성을 배웠습니다. 팀 프로젝트는 단 한명의 실력이 뛰어나다 해서 되는 것이 아닌, 개개인이 여러 의견을 내고 원활한 소통을 통한 협동심이 큰 성과를 내기에 가장 좋은 거름이라는 것을 몸소 느꼈습니다.
개발 과정들을 문서화 하여 보고서를 작성하며 개발자에게 글쓰기 실력도 필요하다는 것을 깨달았던 프로젝트입니다.

(내용 추가) 해당 공모전에서 은상을 수상하는 좋은 결과를 얻었습니다.
