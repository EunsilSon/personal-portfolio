게스트북 : 방명록 웹 사이트

2022.06 - 2022.07 (1人 개인 프로젝트)

‘게스트북’은 방명록을 남길 수 있는 웹 서비스입니다.
기본적인 CRUD 기능을 제공하며, 사용자들은 회원가입을 거친 후 서비스를 사용할 수 있습니다.
Spring Boot 기반의 REST API 서버를 개발하고, CORS Policy를 해결했습니다. 스프링 데이터 JPA를 적용해 DB를 관리하며 트랜잭션의 중요성을 알게됐습니다.
프론트 작업은 VanilaJs를 사용했으며 Axios 라이브러리로 서버와 통신했습니다. AWS EC2 서버를 구축해 Docker를 사용하여 배포했습니다.

▸ 주요 기능

모든 카드 보기, 내가 쓴 카드 보기, 카드 및 댓글 작성 · 수정 · 삭제 · 조회

▸ Front-end

HTML/CSS, Vanila JS, Axios

▸ Back-end

Java, Spring Boot, JPA (Hibernate)

▸ Deployment

AWS (EC2, RDS)

▸ GitHub

https://github.com/EunsilSon/guest-book

▸ 회고록

포트폴리오 웹 사이트

2021.11 - 2021.12 (1人 개인 프로젝트)

지금 보고 계시는 저의 원 페이지 포트폴리오 웹입니다.
프론트엔드 개발 경험이 없어 기본부터 배워보고자 순수 HTML과 CSS로 개발했습니다.
스크롤을 내리며 모든 정보를 볼 수 있고, 상단 메뉴바는 상단에 고정되어 함께 스크롤 됩니다.
모든 HTML파일을 하나의 HTML파일에 불러오기 위해 data-include-path을 사용했고, 출처를 동일하게 인식하기 위해 HTTP Server를 설치해 로컬에서 개발했습니다.
AWS EC2를 구축하고, Apache2를 사용해 배포했습니다.
AWS Route 53을 이용해 도메인을 적용했습니다.

▸ 주요 기능

개인 정보, 보유 기술, GitHub · Blog 링크, 개발 프로젝트 설명

▸ Front-end

HTML/CSS

▸ Deployment

AWS EC2, AWS Route 53, Ubuntu, Apache2

One-Pass

2021.03 - 2021.09 (5人 팀 프로젝트)

‘One-Pass’는 문제은행식의 CCNA 문제가 출제되는 학교 시험을 대비하기 위한 문제 풀이 웹 서비스입니다. Spring Boot 기반의 백엔드 서버는 독립적인 서비스로 분리되어 미들웨어 서버 Eureka를 기준으로 연결됩니다.
Spring Boot 기반의 REST 서버를 개발하고, 서비스들이 분리되어 Eureka 서버를 기준으로 연결되는 MSA 환경을 경험했습니다.
Jenkins와 GitHub WebHook을 연결해 자동 빌드, Docker와 Docker Hub를 이용해 자동 배포를 구현했습니다. Shell script를 작성해 모든 자동화 단계 실행했습니다.

자동화 도구를 사용하지 않고 직접 구현하며 요구 사항에 맞게 시스템을 구축하는 능력과, 여러 기술의 상호 작용에 대한 이해가 깊어졌습니다.

▸ 주요 기능

원하는 개수만큼 문제 풀기, 문제의 핵심 키워드 보기, 포인트 기능

▸ Back-end

Java, Spring Boot, JPA (Hibernate)

▸ Deployment

CentOS, AWS, GCP, Jenkins, Docker, MariaDB

▸ GitHub

https://github.com/Team-Rev

Navo Ground

2020.09 - 2020.11 (5人 팀 프로젝트)

'Navo Ground'는 실시간 멀티 2D 슈팅 게임입니다.
최대 5명이 동시 접속하여 함께 게임을 할 수 있습니다.

클라이언트끼리 실시간 데이터를 주고받을 때 String 타입이 많은 메모리를 점유하여 JSONObject로 파싱 후 통신해 메모리 부하를 해결했습니다.

▸ 주요 기능

상하좌우 이동, 무기 습득, 공격, 무기 버리기, 미니맵 보기

▸ Back-end

Java, JDBC, Netty

▸ Deployment
