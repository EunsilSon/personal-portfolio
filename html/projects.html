게스트북 : 방명록 웹 사이트

2022.06 - 2022.07 (1人 개인 프로젝트)

메세지를 남길 수 있는 방명록 목적의 게스트북 입니다.

실제 운영 가능한 서비스가 아닌 공부 목적의 프로젝트 입니다. 졸업 작품 프로젝트에서 미숙하게 사용했던 기술을 제대로 사용해보기 위해 CRUD에 초점을 두었습니다.
FE와 BE 둘 다 개발하며 REST API의 사용법을 더 숙지한 계기가 되었습니다. JPA의 Hibernate로 DB 작업을 하면서 트랜잭션의 중요성과 영속성 문제를 찾아보게 되었습니다. 서버는 클라우드 서버에 Docker로 배포 했습니다.

발생한 이슈 및 해결 방법과 개발 회고록이 공부 블로그에 기록되어 있습니다.

모든 카드 보기, 내가 쓴 카드 보기, 카드 및 댓글 작성 · 수정 · 삭제 · 조회

Linux 환경의 JSP 웹 사이트 구축

2020.12 (1人 서버 구축 과제)

‘Linux 환경에서 JDK+TOMCAT+MYSQL 을 연동한 사이트 구축’ 과제를 받아 수행했습니다. Ubuntu 운영체제의 터미널에서 작업을 진행했고 JDK, Tomcat8, MySQL을 설치 및 연동해 웹 서버를 구축했습니다.
샘플 페이지 코드를 실행시켜 JDBC 연동, 페이지 실행 및 DB 접근 테스트를 진행했습니다.
CMD 환경에 익숙해지고 리눅스의 폴더 구조를 몸소 느낄 수 있었던 과제입니다.
VMware에서 가상 OS를 설치해 진행했으며 '일단 부딪혀보자' 라는 생각에 여러 명령들을 입력해보고 무언가 꼬였다는 느낌이 들면 가상 OS를 재설치하여 처음부터 다시 작업해보면서 리눅스에 흥미를 갖게되었습니다.

▸ 주요 기능

로그인, 게시글 생성 · 수정 · 삭제

▸ Deployment

Linux, Tomcat8, MySQL

2020 이브와 ICT : 스마트 제어 수직 형 스마트 팜

2020.05 - 2020.11 (5人 팀 프로젝트)

2020 이브와 ICT 공모전에서 은상을 수상한 '수직 파이프 스마트팜' 입니다. 사물인터넷(IoT) 을 통해 작물 성장에 필요한 데이터를 수집하여 장소와 날씨에 구애받지않으며 작물을 재배할 수 있는 ict를 활용한 영농 기술입니다.
팀장을 맡아 전반적인 업무 총괄 및 보고서 작성을 맡았습니다. 스마트팜을 개발하며 팀원들과의 의사소통과 협동심의 중요성을 배웠습니다. 개개인이 여러 의견을 내고 원활한 소통을 통한 협동심이 큰 성과를 내기에 가장 좋은 거름이라는 것을 몸소 느꼈습니다.
개발 과정들을 문서화하며 개발자에게 글쓰기 실력도 필요하다는 것을 깨달았습니다. 상품을 출시하기 위해선 고객에게 우리의 개발품을 쉽게 설명할 수 있어야 합니다. 개발 언어를 비개발 언어로 바꿀 수 있는 능력이 필요했습니다.

재배기 실시간 모니터링 및 온·습도 현황, 조도 · pH · 물 수위 센서
