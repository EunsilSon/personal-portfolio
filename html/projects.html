게스트북 : 방명록 웹 사이트

2022.06 - 2022.07 (1人 개인 프로젝트)

방명록을 남길 수 있는 웹 서비스입니다. 기본적인 CRUD 기능을 제공하며, 사용자들은 회원가입을 거친 후 서비스를 사용할 수 있습니다.
Spring Boot 기반의 REST API 서버를 개발하고, MVC 패턴으로 모듈화하여 코드를 작성했습니다. 스프링 데이터 JPA를 활용하여 객체와 테이블을 매핑하고 연관 관계를 설정했습니다.
Axios 라이브러리를 통한 데이터 송수신 처리를 경험해 클라이언트에서의 API 통신을 이해하고 있습니다. AWS EC2를 구축해 Docker의 Tomcat 컨테이너를 배포했습니다.

▸ 주요 기능

모든 카드 보기, 내가 쓴 카드 보기, 카드 및 댓글 작성 · 수정 · 삭제 · 조회

▸ Front-end

HTML/CSS, Vanila JS, Axios

▸ Back-end

Java, Spring Boot, Spring Data JPA

▸ Deployment

AWS EC2 · RDS, MariaDB

One-Pass

2021.03 - 2021.09 (5人 팀 프로젝트)

문제은행식의 CCNA 시험을 대비하기 위한 문제 풀이 웹 서비스입니다. Spring Boot 기반의 백엔드 서버는 독립적인 서비스로 분리되어 미들웨어 서버 Eureka를 기준으로 연결됩니다.
Spring Boot 기반의 REST 서버를 개발하고, MSA의 포인트 기능과 문의 게시판을 개발했습니다.
Jenkins와 GitHub Webhooks을 연동해 자동 빌드, Docker와 Docker Hub를 활용해 자동 배포를 구현했습니다. Shell Script를 작성해 모든 자동화 단계를 원격으로 실행했습니다.

요구 사항에 맞게 시스템을 구축하는 능력과, 여러 기술의 상호 작용에 대한 이해가 깊어졌습니다.

▸ 주요 기능

원하는 개수만큼 문제 풀기, 문제의 핵심 키워드 보기, 포인트 기능

▸ Back-end

Java, Spring Boot, Spring Data JPA

▸ Deployment

CentOS, AWS, GCP, Jenkins, Docker, MariaDB

Navo Ground

2020.09 - 2020.11 (5人 팀 프로젝트)

'Navo Ground'는 실시간 멀티 2D 슈팅 게임입니다.
최대 5명이 동시 접속하여 함께 게임을 할 수 있습니다.

클라이언트끼리 실시간 데이터를 주고받을 때 String 타입이 많은 메모리를 점유하여 JSONObject로 파싱 후 통신해 메모리 부하를 해결했습니다.

2020 이브와 ICT : 스마트 제어 수직 형 스마트 팜

2020.05 - 2020.11 (5人 팀 프로젝트)

2020 이브와 ICT 공모전에서 은상을 수상한 '수직 파이프 스마트팜' 입니다. '스마트 팜'은 사물인터넷(IoT)를 통해 작물 성장에 필요한 데이터를 수집하여 작물 생산 및 관리를 자동화하는 농업 시스템입니다.
팀장으로서 전체적인 진행 상황을 모니터링하고 작업 조율, 팀원 컨디션 파악, 원활한 소통 등 개발 외적인 부분에도 많은 기여를 하며 리더십을 발휘했습니다.
원활한 협업을 위해선 실력 뿐만 아니라 팀원들과의 의사소통 또한 매우 중요하다는 것을 배웠습니다.
주기적으로 보고서를 작성하며 유지 보수를 위한 기반을 마련했습니다. 개발 과정을 문서화하면서 릴리즈 노트나 에러 로그를 작성할 때에도 다른 사람이 쉽게 이해할 수 있도록 작성해야 한다는 것을 깨달았습니다.

▸ 주요 기능

재배기 실시간 모니터링 및 온·습도 현황, 조도 · pH · 물 수위 센서

▸ Back-end

Arduino, Python

▸ Deployment

Raspberry Pi 4, MySQL
