게스트북 : 방명록 웹 사이트

2022.06 - 2022.07 (1人 개인 프로젝트)

‘게스트북’은 방명록을 남길 수 있는 웹 서비스입니다. 기본적인 CRUD 기능을 제공하며, 사용자들은 회원가입을 거친 후 서비스를 사용할 수 있습니다.
Spring Boot 기반의 REST API 서버를 개발하고, 스프링 데이터 JPA를 사용해 DB를 관리하며 트랜잭션의 중요성을 알게됐습니다. 또, 서버와 다른 출처에 있는 자원을 호출할 경우 발생하는 CORS Policy를 알게됐습니다.
AWS EC2 서버 구축 및 Docker를 사용하여 배포했습니다.

▸ 주요 기능

모든 카드 보기, 내가 쓴 카드 보기, 카드 및 댓글 작성 · 수정 · 삭제 · 조회

▸ Front-end

HTML/CSS, Vanila JS, Axios

Springboot, JPA (Hibernate)
